# "Конструктор"

## Технологии, использованные в проекте

* Vite
* React
* Typescript
* linaria
* react-draggable

## Ссылка на дэплой проекта

[Vercel](https://hs-test-2.vercel.app)

## Запуск проекта

Запуск в режиме разработчика:

```console
yarn dev
```

Cборка:

```console
yarn build
```

Запуск в режиме предварительного просмотра:

```console
yarn preview
```
